¿Qué es el Back-end?

Definición Clave: El back-end es el conjunto de herramientas y lenguajes que procesan las solicitudes de los usuarios, gestionan la lógica, interactúan con bases de datos y generan contenido dinámico, funcionando del lado del servidor.

Importancia:

  • Gestión de la Lógica: Procesa las acciones del usuario.
  • Interacción con Bases de Datos: Almacena y recupera información.
  • Autenticación y Seguridad: Verifica la identidad y controla accesos.
  • Contenido Dinámico: Crea páginas personalizadas.

El Lenguaje de la Web: Protocolos

Definición: Son las reglas que permiten que los dispositivos se comuniquen e intercambien datos de forma correcta y segura.

Django: El Framework para Perfeccionistas

¿Qué es? Un framework de alto nivel para Python que permite crear sitios web complejos de forma rápida, segura y escalable.

Arquitectura MVT (Modelo-Vista-Plantilla)

  • Modelo (`models.py`): La estructura de los datos.
  • Vista (`views.py`): La lógica de la aplicación.
  • Plantilla (`templates/`): La presentación final (HTML).

Comandos Esenciales

Instalar Django:

pip install django

Crear un Proyecto:

django-admin startproject nombre_proyecto

Ejecutar el Servidor:

python manage.py runserver

# Solución al ejercicio de diagnóstico
class Persona:
    def __init__(self, nombre, apellido, edad):
        self.nombre = nombre
        self.apellido = apellido
        self.edad = edad

    def __str__(self):
        return f"{self.nombre} {self.apellido}, {self.edad} años"

def programa_principal():
    lista_personas = []
    try:
        n = int(input("¿Cuántas personas desea ingresar? "))
    except ValueError:
        n = 0

    for i in range(n):
        nombre = input(f"Nombre persona {i+1}: ")
        apellido = input(f"Apellido persona {i+1}: ")
        while True:
            try:
                edad = int(input(f"Edad persona {i+1}: "))
                break
            except ValueError:
                print("Edad inválida.")
        lista_personas.append(Persona(nombre, apellido, edad))

    print("\n--- Listado ---")
    for persona in lista_personas:
        print(persona)

if __name__ == "__main__":
    programa_principal()
